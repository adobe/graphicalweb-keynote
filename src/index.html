About Div

Most of the characters are contained in div elements.

We inject the divs with the contents that make up each character.

About CSS

The fill colors of the CSS character fade from one color to another with CSS transitions.

The eyes and mouths of the characters are also done using CSS animations, which adjust the y-scale.

About SVG

D3.js was used to interpolate paths for the SVG character as well as the curves of the terrain in the scenery.

About 3D Transforms

The scene is managed in 3D space by a root scene element which contains the nested elements transformed in 3D space.

The 2.5D planets and geometry are created by rotating these elements within a single container, which then can be manipulated as one element.

About Canvas

The Canvas character is actually a Canvas element.

Processing.js was used to easily create a particle system and render on 2D Canvas.

About WebGL

The WebGL scene is also a Canvas element utilizing the webgl context.

THREE.js was used to easily create a 3D scene and geometry.

Custom fragment shaders were used to create glowing light effects.

About Blend Modes

For Blend Modes we used overlay to manipulate the color blending of the character with the scenery.

About CSS Custom Filters

A custom vertex shader was written and implemented with CSS Custom Filters to create the fluid effect of a stingray.
